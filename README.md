
# ThriveStack

**ThriveStack** is an AI-powered career growth platform built with Next.js.
It helps users prepare for interviews, generate resumes & cover letters, and receive personalized career insights.

---

## Planned Features

* Authentication & User Onboarding
* AI-Generated Industry Insights
* Mock Interview Generator
* Interview Performance Analytics
* AI Resume Builder (Markdown → PDF)
* Cover Letter Generator
* Weekly Insight Automation (Cron Jobs)

---

## Tech Stack

* **Next.js (App Router)**
* **React**
* **Tailwind CSS**
* **Shadcn UI**
* **Gemini AI API**
* **Inngest**
* **Prisma + PostgreSQL**

---

## Project Status

Currently in development — building core features step by step.
